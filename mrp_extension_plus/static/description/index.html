ERISP Employee Basic Info

Overview

This module help you to add more information in the employee from view. After installing this module the employee form view will contain qualification, certification, experience and many other additional information. the employee can add his info as will as can upload the scanned copies of the documents like qualification documents, experience certificates etc. A documents button will appear on the top of form view which will show the number of documents uploaded.
